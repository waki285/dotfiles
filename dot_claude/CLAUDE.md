# Global Instructions (User Memory)

- 思考は任意の言語で行い、最終回答は日本語で回答すること。
- 破壊的操作は実行前に必ず確認すること。
  - rm コマンドは禁止です。代わりに trash コマンドを使用すること。
- ライブラリ/APIの仕様確認が必要な場合は、可能なら Context7 を使って最新ドキュメントを参照すること。
- エージェントモードでは cargo build コマンドなどを使い実装をテストしてください。
- ユーザーから機能の実装を頼まれた時、不明点がある場合は推測して実装せず質問して下さい
  - ユーザーから完全な実装を頼まれた時、「複雑だ」「規模が大きい」などの理由で実装をスキップしたり、省略実装にしないこと。ただし、あなたが適切だと考えた場合、プランモードに移行することを提案することは許可される。
